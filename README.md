# 弹幕BOSS战 - 年会互动游戏

一款玩家通过发送弹幕攻击BOSS的互动游戏。

## 游戏特色

- 🎮 **弹幕射击**：玩家发送的弹幕会从屏幕左侧慢速飞向右侧
- 👤 **个性化**：每个玩家可以上传头像和设置名字
- 🎯 **BOSS战斗**：BOSS在屏幕中移动，只有击中BOSS的弹幕才会造成伤害
- 💥 **暴击系统**：15%的暴击率，暴击造成2倍伤害
- 🏆 **击杀者展示**：最后一个击杀BOSS的玩家会在屏幕中央被展示

## 游戏玩法

### 1. 玩家设置
- 进入游戏后，首先需要：
  - 上传你的头像（点击"选择头像"按钮）
  - 输入你的玩家名字
  - 点击"确认设置"开始游戏

### 2. 发送弹幕
- 在底部输入框输入弹幕内容
- 点击"发送弹幕"按钮或按Enter键发送
- 弹幕会携带你的头像和名字从左侧飞向右侧

### 3. 攻击BOSS
- BOSS会在屏幕中央区域移动
- 当弹幕碰撞到BOSS时，会造成伤害
- 有15%的几率触发暴击（2倍伤害）
- 战斗日志会实时显示伤害信息

### 4. 击杀BOSS
- 当BOSS血量归零时，最后一个造成伤害的玩家会被标记为击杀者
- 击杀者会在屏幕中央被展示（头像+名字）
- 点击"再来一次"可以重新开始游戏

## 游戏配置

可以在 `game.js` 文件中的 `CONFIG` 对象中调整游戏参数：

```javascript
const CONFIG = {
    BOSS: {
        INITIAL_HP: 500,        // BOSS初始血量
        SIZE: 80,                // BOSS大小
        MOVE_SPEED: 0.6,           // BOSS移动速度
    },
    BULLET: {
        SPEED: 0.6,                // 弹幕移动速度
        BASE_DAMAGE: 10,         // 基础伤害
        CRITICAL_RATE: 0.15,     // 暴击率（15%）
        CRITICAL_MULTIPLIER: 5   // 暴击伤害倍数
    }
};
```

## 技术实现

### 核心系统
- **玩家数据模块**：管理玩家头像、昵称等信息
- **弹幕管理系统**：弹幕的创建、移动、生命周期管理
- **碰撞检测系统**：弹幕与BOSS的碰撞检测
- **BOSS行为系统**：BOSS移动逻辑、受击反馈、死亡事件
- **战斗系统**：伤害计算、暴击机制
- **UI显示系统**：弹幕UI、玩家信息、击杀者展示

### 技术栈
- HTML5 Canvas：游戏渲染
- 原生JavaScript：游戏逻辑
- CSS3：界面样式和动画

## 使用方法

**⚠️ 重要提示：必须使用HTTP服务器运行，不能直接双击打开HTML文件！**

直接打开HTML文件会导致无法加载配置文件（`default-config.json` 和 `version.json`）。

### 启动方式（任选其一）

#### 方式1：使用启动脚本（最简单）
- **Windows**: 双击 `start-server.bat` 文件
- **Mac/Linux**: 在终端运行 `bash start-server.sh`

#### 方式2：手动启动服务器

**使用Python（推荐）:**
```bash
python -m http.server 8000
```

**使用Node.js:**
```bash
npx http-server -p 8000
```

#### 方式3：使用其他HTTP服务器工具
- VS Code的Live Server插件
- 其他任何HTTP服务器工具

### 访问游戏

启动服务器后，在浏览器中访问：**http://localhost:8000**
